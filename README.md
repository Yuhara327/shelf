# はじめに
Shelfは、個人向け蔵書管理ソフトウェアです。  
バーコードの写真から簡単に書籍のデータベースを作成し一覧できます。  
MacOS上で動作します。  

[ダウンロード](https://www.dropbox.com/scl/fi/gw04ehu32w5to6am2duk2/Shelf.zip?rlkey=xa2uwykk262uo5m7ztvgfgzqm&st=80x759ej&dl=1)

# 使用方法
### 1. 書籍を追加する
#### 1.1 ISBNコードを入力
「追加する本のISBNコード」の欄に、追加したい本のISBNコードを入力し、「変更を適用(⌘S)」を押してください。
#### 1.2 バーコードの画像
バーコードの画像を撮影し、一つのフォルダに画像ファイルをまとめてください。「バーコードの画像ファイルから本を追加する」ボタンからフォルダを選択し、本を登録できます。
#### 1.3 Macbook・iphone・Webカメラ等でバーコードを読み取る
「カメラでバーコードを読み取る」ボタンを押すと、バーコードを直接読み取ることができます。
### 2. 書籍を削除する
#### 2.1 ISBNコードを入力
「追加する本のISBNコード」の欄に、追加したい本のISBNコードを入力し、「変更を適用(⌘S)」を押してください。
#### 2.2 チェックボックスで選択
削除したい本の「削除」チェックボックスにチェックを入れ、「変更を適用(⌘S)」を押してください。
### 3. 管理する
「分類番号」の列には、皆さんの好きな文字列を入力することができます。
「既読」チェックボックスで、読んだ本を識別できます。
